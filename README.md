# Team1_Capstone_1_Loan Navigator Agent Suite

## Overview
This Jupyter Notebook implements a multi-agent AI system to automate loan-related customer support queries for a fintech use case. The system routes user queries to specialized agents for database lookup, policy retrieval, and EMI calculations.

---

## Features
- Intent classification using a Supervisor agent
- Natural language to SQL querying on loan data
- Policy-based question answering using a vector database
- EMI and prepayment calculations
- End-to-end tracing using MLflow

---

## Technologies Used
- Python (Jupyter Notebook)
- LangGraph
- Gemini (Vertex AI)
- Chroma Vector Database
- SQLite
- MLflow

---

## How to Run
1. Open the `.ipynb` file in Jupyter Notebook or Google Colab
2. Install dependencies using the provided cells
3. Run all cells from top to bottom
4. Modify the query cell to test different loan-related questions
